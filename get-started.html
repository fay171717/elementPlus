最新 Vue3 + TS 高仿 ElementPlus 打造自己的组件库 ​

安装 ​

bash
npm i @eric-ui --save

开始使用 ​

全局使用

js
// 引入所有组件
import EricUI from "elementPlus";
// 引入样式
import "elementPlus/dist/index.css";

import App from "./App.vue";
// 全局使用
createApp(App).use(ErElement).mount("#app");
vue
<template>
  <er-button>我是 Button</er-button>
</template>

Eric-UI 提供了基于 ES Module 的开箱即用的 Tree Shaking 功能。

vue

  <er-button>我是 Button</er-button>

import { ErButton } from " elementPlus";
export default {
  components: { ErButton },
};
</script>
